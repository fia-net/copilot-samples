Contents:
=========

This package contains working sample code on how to call Certissim Payment
Authentication Management web-service (a.k.a. "copilot") using Java.

WARNING:
As a sample code, it is given solely for evaluating the integration of the
copilot web-service on an external platform and *cannot be used as-is* on
a production platform.

Build system:
=============

The package can be built automatically using Apache MAVEN. Other build system
require manual configuration.


Dependencies:
=============

The source code has been written and tested with the packages and versions
listed hereunder. It may or may not work on other systems and versions, but
there is absolutely no guarantee on that.


** JDK: **
The package has been written for Java7 (OpenJDK 1.7)

** 3rd party libraries: **

- Apache Commons IO 2.4
http://commons.apache.org/proper/commons-io/

- Apache Commons Codec 1.9
http://commons.apache.org/proper/commons-codec/

- Apache HttpClient 4.3
http://hc.apache.org/httpcomponents-client-4.3.x/index.html
